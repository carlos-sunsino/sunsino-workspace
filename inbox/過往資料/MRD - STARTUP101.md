# 一、產品目標與定位

### 1.1 產品目標

打造全球最大、資訊最新、媒合效率最高的新創募資與媒合平台，讓投資人能第一時間掌握最完整、最即時的新創資訊，並協助新創團隊高效獲得資金與資源。

### 1.2 產品定位

- **最新新創資訊優勢**：比其他平台更快上架新創履歷與相關消息，確保投資人優先關注 STARTUP101，掌握第一手投資機會。
- **新創募資履歷平台**：如同新創圈的 104，結合 LinkedIn、AngelList、Dealroom 等國際平台優勢。
- **精準媒合與透明資訊**：協助新創高效獲得資源與資金，讓投資人快速找到潛力標的，創造雙邊網絡效應，提升平台價值。
- **創新賦能資源中心**：提供募資知識、數位工具、專家輔導等一站式服務，幫助新創團隊建立專業形象，吸引更多投資人關注。

# 二、目標市場與用戶分析

### 2.1 目標市場

### 2.1.1 總體潛在市場 (Total Addressable Market, TAM)

- **全球募資工具市場**：全球募資管理軟體市場規模預計將從2024年的24.7億美元增長至2031年的72.9億美元，年複合增長率（CAGR）達到10.3%。其中，智能匹配引擎與盡職調查輔助系統等進階功能的市場，正以高達23%的年複合增長率快速擴張。
- **新創數據情報市場**：此市場展現爆發性成長，以PitchBook為例，其單一平台的TAM已達到65億美元。市場需求包括動態資料追蹤、預測分析服務以及特定產業的垂直情報。
- **整體市場潛力**：綜合來看，新創募資媒合平台的總體潛在市場（包含募資工具與數據服務）估計達到千億美元級別，其中智能工具與數據服務預計將佔據約65%的市場份額。

### 2.1.2 可服務市場 (Serviceable Addressable Market, SAM)

本平台的可服務市場將聚焦於對數據驅動募資解決方案有明確需求的特定客群及區域：

- **地理區域**：初期重點服務亞太地區，特別是台灣（作為戰略樞紐）、印度（AI領域募資活躍）以及東南亞市場（如印尼、越南，早期階段募資集中且監管紅利釋放）。
- **目標客群**：
  - **新創企業（供給端）**：
    - **種子輪前期 (Pre-Seed)**：佔新創企業約38%，主要需求為早期市場曝光及商業計劃書（BP）優化工具。
    - **A輪 (Series A)**：佔新創企業約29%，核心需求為投資者關係管理（IRM）系統。
    - **成長期 (Growth Stage)**：佔新創企業約18%，側重於跨國募資管道的整合。
  - **投資機構（需求端）**：
    - **天使投資人/團**：89%依賴自動化初篩。
    - **CVC**：72%需產業鏈協同篩選。
    - **主權基金/大型機構**：關注地緣政治風險評估、深度技術盡調。
- **核心服務對應**：平台的智能匹配系統、動態盡職調查儀表板、虛擬路演系統等核心功能，旨在滿足上述目標客群在募資與投資決策中的關鍵痛點。

### 2.1.3 可實現市場 (Serviceable Obtainable Market, SOM)

在可服務市場的基礎上，本平台初期可實現的市場將透過以下策略達成：

- **初期市場目標與策略**：
  - **深耕國內市場**：鞏固基礎，作為拓展亞太市場的基石。
  - **台灣樞紐**：鏈結東北亞資源 (日韓Startup)。
  - **差異化競爭**：
    - **深度垂直**：聚焦AI醫療、東南亞清真科技等高潛力領域。
    - **全面本地化**：遵循當地法規、語言及商業習慣。
    - **技術壁壘**：開發AI新創評估等獨特技術。
- **預期成果 (18-24個月)**：在台灣及1-2個東南亞市場的特定垂直領域 (AI、潔淨科技) 獲取初步市場份額，成為領導平台。詳細目標依GTM計畫。

### 2.2 用戶需求

- **新創企業**：
  - **高效募資**：快速對接投資人、降本增效。
  - **專業形象**：履歷/簡報優化、專家諮詢。
  - **市場/資源對接**：產業報告、趨勢分析、夥伴網絡。
  - **知識/技能提升**：課程、工作坊、案例分享。
- **投資機構**：
  - **優質案源**：結構化搜尋、AI推薦、潛力預警。
  - **高效投前管理**：盡調工具、數據驗證、背景核實。
  - **決策輔助**：防範假數據 (52%痛點)、產業鏈協同 (CVC)、地緣政治風險評估。
  - **投後管理/趨勢洞察**：組合管理工具、趨勢報告。
- **生態系合作夥伴**：
  - **資源整合/共享**：案源互推、數據共享、API串接。
  - **共同價值創造**：聯合活動、品牌曝光、生態影響力。
  - **數據洞察/趨勢分析**：共享市場數據、合作產業報告。

# 三、競品分析

### 3.1 國內外主要競品

- **國外**：AngelList (50萬+新創)、Crunchbase (40萬+新創數據)、Dealroom (30萬+新創)、LinkedIn (部分功能)。
- **國內**：Meet創業小聚 (5千+新創)、創業小聚 (3千+新創)、AppWorks (400+新創)、台灣新創競技場 (2千+新創)。

### 3.2 競品優劣

- **國外平台**：資料完整、國際資源多，但本地化不足、語言與法規障礙。
- **國內平台**：社群活躍、活動多，但缺乏結構化募資履歷、媒合與數位工具整合。

### 3.3 STARTUP101 獨特賣點(USP)

- **快速募資履歷，效率提升70%**：新版更簡明，填寫省 70% 時間，幫助新創快速建立專業形象，新創履歷平均編輯至上架時間僅需2天。
- **第一手新創資訊，領先市場**：比競品更快上架新創履歷與消息，確保投資人第一時間掌握市場動態，提升市場影響力與用戶信任度，成為首要關注平台。
- **精準媒合，省時50%以上**：領域標籤、偏好推薦、CRM 功能，讓投資人精準找到潛力標的，節省案源搜尋時間，創造雙邊網絡效應驅動新創與投資人互動。
- **資訊透明，降低調查成本**：建立信任基礎，透過身份驗證與透明化流程，減少投資人盡職調查負擔。
- **數位工具，募資成功率增20%**：課程、簡報模版、數據洞察，協助新創提升募資成功率。
- **線上線下整合，創造實質連結**：媒合會、論壇、顧問服務，促進新創與投資人間的商業合作。

# 四、產品功能需求

### 4.1 新創端

- **募資履歷建立**：AI履歷引導撰寫、新創履歷展示模板、投資人10大問題模板、歷史募資資訊歷程、外部資訊整合。
- **資源媒合**：領域標籤、閉門媒合會、加速器/競賽/論壇活動對接。
- **投資者關係管理（IRM）**：投資人溝通追蹤、投資報告產出、投資人名單管理、會議安排協助。
- **數位賦能**：募資知識課程、簡報模版、財報/財測工具、用戶數據洞察。
- **曝光與行銷**：新創故事、專訪、電子報、活動推廣。

### 4.2 投資人端

- **新創搜尋與篩選**：AI搜尋、偏好推薦、CRM管理。
- **盡職調查工具**：資料室（Data Room）、身份驗證、電子簽章。
- **趨勢洞察**：白皮書、趨勢分析、專家內容。

### 4.3 平台共通

- **會員系統**：新創/投資人/合作夥伴多角色註冊與管理。
- **活動管理**：線上/線下活動報名、管理、媒合追蹤。
- **數據分析**：用戶行為、媒合效率、募資進度等儀表板。
- **API整合**：與外部加速器、政府、國際平台串接。
- **點數機制**：促進活耀度(每日登入/分享履歷/邀請註冊)、誘因(解鎖履歷)，促使解鎖與分享效率提升。

# 六、商業模式與成長策略

- **目標市場細分**：初期聚焦台灣市場，針對不同成長階段的新創團隊以及本地各類型投資人，建立首批核心用戶群，奠定市場基礎。
- **核心價值主張**：強調平台在募資履歷建立上的高效便捷、資訊更新的領先速度，以及精準媒合的顯著優勢，吸引新創與投資人積極參與。
- **定價與試用方案**：為新創團隊提供初期免費試用會員功能，結合靈活的訂閱制與加值服務模式，並為投資人提供短期免費體驗，降低進入門檻，提升參與意願。

# 八、風險與挑戰

- **新創與投資人雙邊市場冷啟動難度**：初期透過業務部與拓展育成中心、加速器等合作夥伴，獲取更早期團隊資訊；營運部主動關注平台優質新創，推薦合適投資人互動，促進早期成功案例。
- **用戶假冒身份**：透過「藍勾勾」身份驗證減少假冒問題，提升平台信任度。
- **新創募資知識不足**：新創團隊往往對募資相關知識了解不足，難以提供投資人真正需要的資訊。營運部與行銷部合作推出線上課程與平台乾貨內容等，提升新創能力。
- **數位工具轉型困難**：原因包括對既有方式的習慣難以調整適應，例如募資方式多依賴聚會或地推認識投資人，對於轉向數位平台感到不適應。提供直觀易用的界面、逐步引導的教學流程，以及線上客服支持，幫助用戶快速熟悉平台功能。

- **持續優化媒合引擎與數據分析**：透過用戶行為與媒合效率數據洞察，持續改善媒合演算法，提升匹配精準度，並強化數據分析功能，提供更深入的市場洞察。

- **擴展國際合作與品牌影響力**：與亞太及全球創新生態系合作，參與國際新創活動與展覽，提升STARTUP101品牌知名度，吸引國際用戶。
- **發展AI自動化推薦與募資工具**：利用AI技術自動分析新創資料與投資人偏好，提供更精準的推薦，並開發自動化募資工具，協助新創維繫投資人關係，提升募資效率。
